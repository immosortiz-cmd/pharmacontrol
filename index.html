OneDrive Sync

Sincroniza tu base de datos entre PC, celular y tablet

Estado de conexión
No conectado
Conecta tu cuenta Microsoft para sincronizar datos entre dispositivos
Configuración Azure
Necesitas registrar una app gratuita en Azure para obtener el Client ID. Lee el archivo ONEDRIVE_CONFIGURACION.md incluido en el ZIP.
Encuéntralo en Azure Portal → App Registrations → tu app → Overview
Guardar en OneDrive

Sube el estado actual al OneDrive. Úsalo al terminar de trabajar en este dispositivo.

Cargar desde OneDrive

Descarga los datos desde OneDrive. Úsalo al empezar en un nuevo dispositivo.

Pasos de configuración
1
Registrar app en Azure (gratis)
Ve a portal.azure.com → App registrations → New registration. Tipo: Single-page application.
2
Copiar el Client ID
En Overview copia el "Application (client) ID" y pégalo en el campo de arriba.
3
Agregar Redirect URI en Azure
En Authentication → Redirect URIs → SPA, agrega la URI que aparece en el campo de arriba. Copia el botón 📋.
4
Conectar y sincronizar
Clic en "Conectar con Microsoft", inicia sesión y autoriza. Los datos se guardan en OneDrive/Apps/PharmaControl/data.json
